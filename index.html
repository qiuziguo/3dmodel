🏮 关公像 AR 体验

在您的桌面上放置一尊威武的关公像

使用说明：

• 点击按钮启动 AR

• 将手机对准桌面或平面

• 点击屏幕放置关公像

• 可以多次点击放置多个雕像

📁 模型设置：

✅ 已配置 EinScan Pro HD 关公像模型
📍 来源：SHINING 3D (Sketchfab)
🔗 GitHub: qiuziguo/3dmodel

重要：需要启用 GitHub Pages
1. 前往您的 GitHub 仓库设置
2. 滚动到 "Pages" 部分
3. 选择 "Deploy from a branch"
4. 选择 "main" 分支
5. 点击 "Save"
